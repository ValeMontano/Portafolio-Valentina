================================
  CV / Currículum Vitae (PDF)
================================

Coloca tu archivo PDF de currículum en esta carpeta con el nombre:

  cv.pdf

El enlace de descarga en el footer de la página apunta a:

  assets/docs/cv.pdf

Una vez que coloques el archivo aquí, el botón "Descargar CV en PDF"
funcionará automáticamente.
